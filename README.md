ArrowFlight - сервер

Сервер для работе на кластерах Apache Ozone и Hadoop,
который осуществляет высокоскоростную передачу данных
по сети в колоночном формате arrow, не зависящем от
формата файлов, в котором хранятся данные. Должен
позволить выполнять часть операций, таких как projection,
filter-pushdown, aggregation локально на сервере, где
хранятся данные, а затем передать промежуточный
результат по сети в формате Arrow движкам Spark, Trino (Java)
и StarRocks. В проекте нужно разработать как серверную часть
(преимущественно Java), так и плагины для Spark, Trino (Java)
и StarRocks (C++).
