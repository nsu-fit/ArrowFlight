# ArrowFlight-сервер

Сервер для работы на кластерах **Apache Ozone** и **Hadoop**, который осуществляет высокоскоростную передачу данных по сети в колоночном формате **Apache Arrow**, не зависящем от формата файлов, в котором хранятся данные.

Сервер должен позволять выполнять часть операций локально на сервере, где хранятся данные, а затем передавать промежуточный результат по сети в формате **Arrow** движкам **Spark**, **Trino** и **StarRocks**.

## Поддерживаемые операции

- **Projection**
- **Filter Pushdown**
- **Aggregation**

## Интеграции

Промежуточный результат должен передаваться в формате **Apache Arrow** следующим движкам:

- **Spark**
- **Trino**
- **StarRocks**

## Разрабатываемые компоненты

В проекте нужно разработать:

- серверную часть, преимущественно на **Java**;
- плагины для **Spark** и **Trino** на **Java**;
- плагин для **StarRocks** на **C++**.
